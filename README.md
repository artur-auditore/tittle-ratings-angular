# Tittle Ratings (Angular)

O projeto foi gerado com [Angular CLI](https://github.com/angular/angular-cli) na versão 8.3.21.

## Development server
Antes de tudo, no terminal execute "npm install"; logo em seguida, "ng serve" e no navegador acesse http://localhost:4200/. A aplicação irá recarregar automaticamente se você fizer mudanças

Este projeto foi feito para consumir [esta API](https://github.com/artur-auditore/movie-ratings).
